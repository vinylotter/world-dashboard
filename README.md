# Wealth & Health Around the World

Interactive D3 dashboard exploring the relationship between life expectancy and GDP per capita.

## Data
Source: Our World in Data (OWID)
- Life expectancy
- GDP per capita (World Bank)

Data was preprocessed with Python to keep the most recent year where both metrics exist for each country.

## Views
- Histogram: life expectancy
- Histogram: GDP per capita
- Scatterplot: GDP per capita vs life expectancy
- Choropleth world map with metric toggle
